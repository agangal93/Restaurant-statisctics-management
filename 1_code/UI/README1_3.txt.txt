This folder contains the code for creating Manager's UI.

To run this code, download XAMPP server - https://www.apachefriends.org/index.html
This will setup a myPHP server on your machine i.e localhost. Copy all files into "htdocs" folder in installed xampp directory. 
Open the XAMP GUI and start Apache and MySQL modules(this will setup your localhost on 3306 port)
Open a web browser and search localhost/login.php. This will open the Manager's login portal.
