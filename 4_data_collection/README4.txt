Data for our has been retrieved from reliable online sources. 
There are 4 CSV files with independent data division for each classification category
1) StateFood.csv - Food type vs StateFood
2) TimevsCustomer.csv - Time vs Customer Density
3) Ethnicity.csv - Location vs Ethnicity
4) FoodList.csv - Food Category vs Ingredient
5) Ingredient.xls - Ingredient count

These files will be used by the code for Data analytics
